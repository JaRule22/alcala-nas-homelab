# alcala-nas-homelab
Alcala NAS homelab is a self-built storage and media server designed for high-performanced streaming and containerizer applications.

## Hardware
- 2x 18TB HDD
- 2x Samsung 990 Pro NVMe SSD
- 2x Samsung 16GM DDR5 RAM
- UGREEN NAS DXP4800 Plus System

## Stack
- Jellyfin media server
- Docker-based services
- SSD-based cache & transcoding layer

## Architecture
- HDD: media storage (Movies, TV)
- SSD: Application data, cache, and transcoding
