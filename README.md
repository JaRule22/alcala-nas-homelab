# alcala-nas-homelab
A self-built home lab environment designed for high-performance storage, media streaming, and containerized application deployment. This project serves as a hands-on learning platform for systems administration, Docker containerization, and network infrastructure.

## Hardware 🖥️
- 2x 18TB HDD
- 2x Samsung 990 Pro NVMe SSD
- 2x Samsung 16GM DDR5 RAM
- UGREEN NAS DXP4800 Plus

## Stack 🛠️
- Jellyfin - self-hosted media server
- Docker - Containerized service deployment
- SSD-based cache & transcoding layer

## Architecture 🏗️
- HDD: media storage (Movies, TV)
- SSD: Application data, cache, and transcoding

## Skills Demonstrated 📚
- Docker containerization
- Systems & network administration
- Storage architecture and management
- Linux environment configuration
- Self-hosted infrastructure deployment
